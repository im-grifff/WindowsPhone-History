  • Windows 10 Mobile

    Ini adalah seri terakhir dari OS Windows Phone. OS Ini bertujuan untuk menyatukan dan memberikan konsistensi yang lebih besar dengan rekan desktop Windows 10-nya. Seri Ini menyediakan Universal Windows Platform (UWP) baru yang memungkinkan aplikasi dapat dijalankan di semua perangkat Windows 10, baik desktop, seluler, atau konsol.

    Codename Threshold

    Released Nov 20, 2015

    End of life Jun 11, 2019

    Build On Windows NT

    • 2017

  • Windows Phone RIP

    Pada bulan Oktober 2017, wakil presiden perusahaan Microsoft, Joe Belfiore, menegaskan bahwa, karena pangsa pasar yang rendah dan kurangnya pengembangan pihak ketiga, Microsoft tidak akan lagi menjual atau memproduksi perangkat Windows 10 Mobile baru.

    So long, Windows Phone. We will miss you.
